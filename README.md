## damnusb

Automates the management of *udev* rules for USB ports and devices


### Build
```sh
    sudo git clone https://github.com/osipenco/damn-usb
    cd damn-usb
    sudo make install
```

### Dependencies
```sh
    # Arch
    sudo pacman -S usbutils
    
    # Debian
    sudo apt install usbutils
    
    # Fedora
    sudo dnf install usbutils
```
